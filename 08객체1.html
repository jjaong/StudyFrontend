객체

Key(속성) + Value(속성값)으로 한쌍으로 저장할 수 있는 구조
여러 속성을 하나의 변수에 저장할 수 있도록 해주는 데이터타입

=> 자바스크립트에서의 배열 : 자바 ArrayList와 비슷한 개념
=> 자바스크립트에서의 객체 : 자바의 HashMap과 비슷한 개념
-객체는 중괄호{}를 사용해서 생성하고,
중괄호 안에 이 객체에 필요로 하는 속성(property) 들을 정의함 (속성 : 속성값) ==(키: 벨류)
참고로 속성값으로는 모든 자료형의 값을 담을 수 있음 

        [표현법]
        var 변수명(==객체명) = {
            속성명 : 속성값,
            속성명 : 속성값,
            속성명 : 속성값,
            .....
        }

with : 코드를 줄여주는 키워드
