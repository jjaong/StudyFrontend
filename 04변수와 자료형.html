변수와 자료형

변수 선언 시 var뿐만 아니라 let, const를 이용해서 선언 기록

* var, let, const의 차이점

1)변수 선언 방법

2)Scope(사용 가능 범위)

*변수 선언 위치에 따른 전역 변수/ 지역변수

        <script>

             var 변수명;
             변수명;

             function 함수명{
                 var 변수명;
                 변수명;

             }

        </script>

*자료형

변수 선언 시 자료형을 별도로 지정하지 않음(자료형의 개념이 없는건 아님)
=> 변수에 대입되는 값(리터럴)에 따라서 알아서 자료형이 자동으로 결정됨 : 타입추론

자바스크립트의 자료형

  • string(문자열)
  • number(숫자)
  • boolean(논리값)
  • object(객체)
  • function(함수)
  • undefined(초기화가 안된 변수)
