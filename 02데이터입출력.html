자바스크립에서의 데이터 입/출력

1. 데이터를 출력하는 기본 구문

1)window.alert('알림창에 출력할 문구');
2)window.console.log('콘솔창에 출력할 문구');
3)document.write('화면에 출력할 문구');
***** 4) 선택한 요소 .innerHTML/innerText = '해당 요소에 출력할 문구';

3) document.write('화면에 출력할 문구');

4.선택한 요소.innerHTML/ innerText = '해당 요소에 출력할 내용';

innerHTML과 innerText 자바스크립트에서 어떤 요소(Elemnet)안의 값(Content)을 가지고 온다거나
변경하고자 할 때 innerHTML, innerText라는 속성에 접근해야한다.
=> 가지고 올 때 : 변수 = 선택한 요소 .innerHTML/ innerText;
=> 변경하고자 할 때 : 선택한 요소.innerHTML/ innerText = '변경할 값';

2. 데이터를 입력받는 기본 구문

1) 변수 = window.confirm('질문 내용');
2) 변수 = window.prompt('질문내용');
3) 변수 = 선택한 요소.속성;(id,className, innerHTML, innerText); 4)변수 = 선택한input요소.value

1) window.confirm('질문내용');

confirm 호출 시 "질문 내용"과 "확인/취소" 버튼이 존재하는 알림창 발생
확인 버튼 클릭시 true, 취소 버튼 누를시 false값 반환

2)window.prompt('질문내용');

promt호출 시 '질문내용'과 입력할 수 있는 텍스트 상자 와 확인/취소 버튼이 보여지는 알림창 발생 -확인 버튼 클릭 시 텍스트 상자에 입력되어있는 값이 반환
-취소 버튼 클릭 시 null값을 반환

    4)선택한 input 요소.value;
